# Boid CLI

> A linux command line interface for Boid 

### Setup

> install this package first

```shell
$ npm install boidcmd -g
```

> now setup boid 

```shell
$ boidcmd setup
```

> additional commands can be found in the help menu

```shell
$ boidcmd help
```
